# tmux Cheatsheet

プレフィックス: `C-a` (Ctrl+a)

## セッション

| 操作 | コマンド/キー |
|------|-------------|
| 新規セッション | `tmux new-session -s <name>` |
| セッション一覧 | `tmux ls` |
| アタッチ | `tmux attach -t <name>` |
| デタッチ | `C-a d` |
| セッション切替 | `C-a s` |
| セッション終了 | `tmux kill-session -t <name>` |

## ウィンドウ

| 操作 | キー |
|------|------|
| 新規ウィンドウ | `C-a c` |
| 次のウィンドウ | `C-a n` |
| 前のウィンドウ | `C-a p` |
| 番号で移動 | `C-a 1` `C-a 2` ... |
| ウィンドウ名変更 | `C-a ,` |
| ウィンドウ一覧 | `C-a w` |
| ウィンドウ閉じる | `C-a &` |

## ペイン

| 操作 | キー |
|------|------|
| 左右分割 | `C-a \|` |
| 上下分割 | `C-a -` |
| ペイン移動 | `C-a h/j/k/l` |
| ペイン移動(マウス) | クリック |
| ペインリサイズ | 境界線ドラッグ |
| レイアウト切替 | `C-a Space` |
| ペイン閉じる | `C-a x` |
| ペイン最大化/復元 | `C-a z` |

## コピーモード

| 操作 | キー |
|------|------|
| コピーモード開始 | `C-a [` |
| スクロール | 矢印キー / マウスホイール |
| 検索(前方) | `/` |
| 検索(後方) | `?` |
| 選択開始 | `Space` |
| コピー | `Enter` |
| ペースト | `C-a ]` |
| コピーモード終了 | `q` |

## その他

| 操作 | キー/コマンド |
|------|-------------|
| 設定リロード | `tmux source ~/.tmux.conf` |
| コマンドモード | `C-a :` |
| キーバインド一覧 | `C-a ?` |
| 全ペインに同時入力 | `C-a :` → `setw synchronize-panes on` |
