# dotfiles

## Requirements

- [Nix](https://nixos.org/)
- [Home Manager](https://github.com/nix-community/home-manager)

## Setup

```bash
# clone
gh repo clone HikaruEgashira/dotfiles
nix run home-manager -- -f ~/dotfiles/home.nix switch
```
